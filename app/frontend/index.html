Verify button mapping

The index → physical button correspondence is auto-detected from your mouse's current button actions, not hardcoded — but it can be wrong if you've already remapped buttons away from their factory defaults. To confirm a marker for yourself:

  1. Pick a button whose identity you want to confirm.
  2. Set its action to Keyboard key → a key you don't normally use (e.g. KEY_F13).
  3. Run xev or showkey in a terminal, then physically press that button on the mouse.
  4. Whichever marker's key shows up is the physical button you pressed — relabel it below if it doesn't match.
